# Comandos do projeto — CareCenterLab

Comandos oficiais do laboratório (já validados na linha de base).

- **Build:** `dotnet build`
- **Testes:** `dotnet test`
- **Rodar API:** `dotnet run --project CareCenterLab.Api`

**Stack:** .NET 8, ASP.NET Core Web API, xUnit.

**Domínio fictício:** centro de cuidado com residentes, medicamentos, sinais vitais e alertas.
Não usa dado real e não representa nenhum cliente — é um domínio de estudo.
